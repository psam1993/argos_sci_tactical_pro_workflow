# Argos SCI Tactical Pro Workflow

## Descrição do Projeto

O projeto **Argos SCI Tactical Pro Workflow** tem como objetivo a construção e padronização de um único formulário para o preenchimento das informações importantes relacionadas aos papéis estabelecidos no Sistema de Comando de Incidentes (SCI). Este sistema visa:

- Obter informações cruciais de forma centralizada.
- Reduzir o preenchimento redundante de dados.
- Facilitar a visualização das ações e do posicionamento das equipes em trabalho por meio de um mapa interativo.

Com isso, o projeto permite uma gestão mais eficiente e um acesso mais rápido às informações, proporcionando suporte aos níveis de decisão.

## Funcionalidades

- **Centralização de Informações**: Um único formulário para consolidar dados importantes.
- **Redução de Redundância**: Evita o preenchimento repetitivo de informações.
- **Visualização em Mapa**: Permite acompanhar as ações e o posicionamento das equipes em tempo real.
- **Apoio à Decisão**: Informações rápidas e acessíveis para suporte aos gestores.
